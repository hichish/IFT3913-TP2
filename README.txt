Pour executer le les metrique vous aller juste utiliser une ligne de commande
Si vous utiliser un Mac vous executer le fichier analysis.sh comme suit:
  ./analysis.sh (vous devez etre dans le projet IFT3913-TP2)
Si vous utiliser windows vous executer le fichier analysis.bat comme suit:
  ./analysis.bat
Pour executer les jars individuellement il faut faire par exemple:
java -jar tDocumente.jar "/Users/ahmadbary/IFT3913-TP2/jfreechart-master"
ou avec le chemin relatif 
java -jar tDocumente.jar ./jfreechart-master
